# bottle-plotter
The code for the buriak group data plotter

## dependencies
- python 3.5+
- bottle
- wtforms
- numpy
- pandas
- matplotlib
- seaborn
- uncertainties

See it in action at:
https://maverick.chem.ualberta.ca/plot/ash

# Run it!
Tested most recently on python 3.8

`pip3 install -r requirements.txt`
`python3 bottle_plot.py`
